# CodeGuardian AI

## 🚀 Autonomous AI-Powered Code Review Platform

CodeGuardian AI is an intelligent multi-agent code review platform that analyzes GitHub pull requests, detects bugs, security vulnerabilities, performance bottlenecks, and generates human-like review comments in real time.
## 🔥 Features
- AI Pull Request Review
- Security Vulnerability Detection
- Performance Optimization Suggestions
- Architecture Analysis
- Risk Scoring
- AI Quick Fix Patch Generation
- Export Reports (PDF/Markdown)
- Multi-Agent Review Workflow
- Repository Analytics Dashboard
